Selectores en CSS

En CSS, los selectores son las herramientas que utilizamos para seleccionar los elementos HTML a los que queremos aplicar estilos. Por ejemplo, podemos cambiar el color de todos los párrafos, de un elemento específico, o de los elementos dentro de una sección.

¿Cómo se usan?

×

Los selectores se usan para decirle al navegador qué elementos HTML deben recibir estilos. Se escriben en CSS seguidos de llaves {}, y dentro se colocan las propiedades, por ejemplo:

p { color: red; }

¿Cuántos tipos existen y para qué sirven?

×

Existen selectores como:

Tipo de Selector Ejemplo ¿Qué selecciona?
De tipo p Todos los elementos <p>
De clase .rojo Elementos con class="rojo"
De ID #titulo Elemento con id="titulo"
Universal * Todos los elementos
Descendiente div p Todos los <p> dentro de un <div>
Hijo directo ul > li Los <li> que son hijos directos de un <ul>
De atributo input[type="text"] Campos de texto
Pseudo-clase a:hover Enlaces cuando el mouse pasa encima
