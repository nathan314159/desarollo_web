¿Qué se puede hacer con CSS?

Antes de 2010, los sitios web estaban diseñados principalmente para pantallas grandes. Con el auge de los dispositivos móviles surgieron varios problemas:

  • Las páginas no se veían bien en pantallas pequeñas.
  • Era necesario hacer "zoom" y desplazarse horizontalmente.
  • La experiencia de usuario era muy pobre.

Solución: se necesitaba una forma de hacer que una misma web funcionara bien en cualquier dispositivo: móviles, tabletas, laptops y monitores.

¿Desde cuándo existe?

El término "Responsive Web Design" fue propuesto por Ethan Marcotte en mayo de 2010, en un artículo de la revista A List Apart. Luego lo desarrolló más profundamente en su libro de 2011.

¿Cómo funciona?

  • Uso de unidades flexibles como %, vw, vh, em.
  • Flexbox y CSS Grid para crear estructuras dinámicas.
  • Imágenes y texto que se ajustan automáticamente.
  • Uso de media queries para adaptar estilos según el tamaño del dispositivo.

Impacto actual

  • Más del 50% del tráfico web proviene de móviles.
  • El diseño responsive es ahora un estándar obligatorio.
  • Google penaliza sitios que no se adaptan a móviles.
